**Opdracht 2** — Resource group `ODISEE_NETWORK` in France Central. `VNET2` (172.16.0.0/16) met SUBNET_01 (172.16.1.0/24) en SUBNET_02 (172.16.2.0/24). NSG `NSG_RDP` voor RDP-toegang. VNet Peering geconfigureerd.
