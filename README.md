# VMware ESXi — Virtualisatie & Active Directory

> **Schoolproject** — Odisee Campus Brussel · Toegepaste Informatica · Cybersecurity & Network Specialist · 2024–2025

Opzetten van een gevirtualiseerde serveromgeving met VMware ESXi, inclusief Windows Server 2022 en Active Directory domeinbeheer.

---

## Technologieën

| Domein | Technologie |
|--------|-------------|
| Hypervisor | VMware ESXi |
| OS | Windows Server 2022 |
| Directoryservice | Active Directory Domain Services (AD DS) |
| DNS | Windows DNS Server |
| DHCP | Windows DHCP Server |
| Groepsbeleid | Group Policy Objects (GPO) |
| Beheer | Remote Desktop, Active Directory Users & Computers |

---

## Behandelde onderwerpen

- **VMware ESXi**: installatie en configuratie van de hypervisor, aanmaken en beheren van virtuele machines
- **Windows Server 2022**: installatie, basisconfiguratie, rollen en features toevoegen
- **Active Directory**: domein opzetten (AD DS), domeincontroller promoveren, gebruikers en groepen aanmaken
- **DNS & DHCP**: geïntegreerde DNS-zones en DHCP-scopes binnen het AD-domein
- **GPO**: groepsbeleid toepassen op OUs voor gecentraliseerd configuratiebeheer
- **Gebruikersbeheer**: OUs, security groups, gebruikersaccounts en wachtwoordbeleid

---

## Omgeving

- Bare-metal VMware ESXi hypervisor
- Meerdere VMs: domeincontroller + workstations
- Intern netwerk met statische IP-configuratie
